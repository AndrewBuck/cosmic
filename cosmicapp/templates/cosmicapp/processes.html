Processes

This page provides a listing of all the different types of task queue processes as well as a brief summary of each one. For a more detailed description, click the name of the specific process below. These tasks may be created by the website itself, or when images or other files are uploaded to Cosmic. The tasks are taken out of the queue in order of priority and then executed.

astrometryNet - The Astrometry.net plate solving program. The plate solver is fed a custom generated list of detected sources from the starmatch task.
generateThumbnails - Runs the ImageMagick 'convert' program to generate a series of 8-bit png images in 4 different sizes suitable for display on the website.
imagestats - Runs the ImageMagick 'identify' program to record the width, height, color channels, fits/png headers, image plate solution, as well as the mean, median and standard deviation of the image and also just for the image background.
parseHeaders - Reads the database entries containing raw image headers and produces a second table of 'cleaned' header tags which are more suitable to be queried on the website.
sextractor - Runs the Source Extractor program to detect both stellar and extended bright objects in an uploaded image.
image2xy - Runs the image2xy program to detect stellar objects in an uploaded image.
daofind - Runs the daofind algorithm from Astropy to detect stellar objects in an uploaded image.
starfind - Runs the starfind algorithm from Astropy to detect stellar objects in an uploaded image.
starmatch - Looks at the results of every image source detection algorithm we run and compares them to eachother to produce a list of 'matches' which represent the same star detected by each separate algorithm as a single result.
